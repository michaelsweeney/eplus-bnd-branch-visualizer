About BND·VIZ

A browser-only EnergyPlus HVAC explorer. It reads the system topology from a run's eplusout.bnd report, 3D zone geometry from the epJSON model, and node temperatures and flows from eplusout.sql, and shows them as synchronized graph, 3D, and chart views over one annual timeline. Everything runs client-side; nothing is uploaded.

github.com/michaelsweeney/eplus-bnd-branch-visualizer

What it reads

Drag files onto the app; each layer degrades gracefully. A .bnd alone gives the full topology graph — every EnergyPlus run writes one, any version, no preparation. Add the epJSON for 3D zone geometry. Add a playback JSON for the temporal layer.

Bring your own model

Only the playback layer needs the model prepared before the run: it requires the System Node Temperature and System Node Mass Flow Rate output variables and Output:SQLite. The CLI helpers (npm run prep and npm run export:playback) inject those outputs, optionally run the version-matched engine, and convert the resulting eplusout.sql into the playback JSON the app loads. See the repo README for the full steps.
